## Frontend:
<br />
React


## Getting Started:

```gem install bundler```

```bundle install```

```rake db:migrate```

```rake db:seed```

```rails s -p 3001```

To run the tests:
```bundle exec rspec```


## Start the frontend server:

```cd client```

```yarn install```

```yarn start```
